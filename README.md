
# Code Snippet Hub

A microservices architecture-based platform for sharing, saving, and commenting on code snippets. Users can create titled snippets, view a list of all snippets, add comments, and persist data locally.



## Features
- Title & Code Input: Simple form for snippet title and code content

- Snippet List: View all snippets with title, code preview, and comments

- Comments: Add comments to any snippet

- Local Storage: All data persists in browser (no backend required)
